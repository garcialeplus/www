Working with Animate

Using Animate was really complex. Developing animations involves very different skills than Photoshop or Illustrator. Carrying out a project in Animate implies having a clear idea of the movements you want, keeping in mind that there is a scenario with a specific order when programming the frames. For me, using this application implied a challenge because it was not easy to develop my ideas. However, in the end, I liked the result of the work that I did, Still I do not believe that animation is where I want to focus my career.
